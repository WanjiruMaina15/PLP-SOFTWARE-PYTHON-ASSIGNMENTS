➤ A CSS rule consists of:

  1. Selector: Targets an HTML element.
  2. Property: Defines what to style.
  3. Value: Specifies the styling applied

CSS Box Model

✨The CSS Box Model describes how every element is structured on a page:

  1. Content: The inner part containing text or other elements.
  2. Padding: Space between the content and the border.
  3. Border: Surrounds the padding.
  4. Margin: Outer space between the element and others.
